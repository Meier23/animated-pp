# App Policy Documents

This repository hosts the official policy and compliance documents for Home Assistant - Unofficial App.

It is used as the public source of truth for store submissions and user-facing legal information.

## Purpose

- Publish a stable, public Privacy Policy URL for app stores
- Keep policy updates transparent with clear dates
- Store supporting compliance documents in one place

## Store Usage

- Google Play: Use the public Privacy Policy URL in Play Console
- Microsoft Store: Use the same Privacy Policy URL in Partner Center

## Update Process

1. Edit the relevant policy document
2. Update the Last updated date in that document
3. Commit and push changes to the default branch
4. Confirm the GitHub Pages URL is live
5. Verify store listing links still point to the same stable URL

## Scope

These documents apply to Home Assistant - Unofficial App releases distributed through app stores, including Android and Windows variants where applicable.
